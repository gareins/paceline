Help

Use cryptographic functions to generate your password, based on username, website data and your personal password.

Hash functions use input string, and output random looking string. Input string is generated using your master password, username and website data.

The way that input string is generated is determined in settings. The input string is determined from "content" field, which determines how your data is combined and inputed in Hash function. Hash function can also be chosen from number of possibilities listed in settings.

Because hash functions by themselves return only raw bitstream, there has to be a method, to generate valid ASCII string. We currently only support two methods:

  • HAX[0:length]
  • Base64[0:length]
In Base64 option, invalid non digit or letter characters are removed. If Resulting string is too short, we append zeroes. Also, length can also be determined in settings.

Example
  • website: got.io
  • username: john.snow
  • password: iknownothing
  • content: '[site.url]|&[uname][pass]'
  • function: SHA512
  • bit2str: Base64
  • length: 16

generated password: 'wDJwpKEd3KNM8JZn'

Currently supported website data is only its url, but more could be added in future.
